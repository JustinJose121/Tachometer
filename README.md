# Tachometer	
Developed a device using Arduino UNO open-source Microcontroller board based on the Microchip ATmega328P which can calculate objectâ€™s revolving speed.

## Getting Started

How to use
```    
git clone https://github.com/aakashjhawar/Tachometer.git
cd Tachometer
```
(Run the ```tachometer.ino``` file) 

## Prerequisites

- Arduino IDE


[Get Arduino IDE](https://www.arduino.cc/en/main/software)
- basic understanding of Arduino UNO

## Components Used
1. IR Pair
2. Arduino UNO
3. Seven Segment Display
